JavaScript正则中\1\2的作用

一、示例

1. 验证6个相同的数字

var reg = new RegExp(/^(\d)\1{5}/g);
var a = '333333';
if(reg.test(a)) {
    alert('right');
} else {
    alert('wrong');
}

2. 验证3个相同的数字+3个相同的字母

var reg = new RegExp(/^(\d)\1{2}([a-z])\2{2}/g);
var a = '333aa1';
if(reg.test(a)) {
    alert('right');
} else {
    alert('wrong');
}

二、解释：

\1必须与小括号配合使用。

正则表达式中的小括号"()"。是代表分组的意思。 如果再其后面出现\1则是代表与第一个小括号中要匹配的内容相同。Link
